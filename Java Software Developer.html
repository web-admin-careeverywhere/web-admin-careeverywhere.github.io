JAVA SOFTWARE DEVELOPER

GENERAL SUMMARY:

The software engineer will be responsible for design and development of functional enhancements of Company’s Medical Device Server software systems, which functions as a middleware wireless connectivity solution of medical device fleet to hospital Electronic Medical Record systems.

RESPONSIBILITIES:

Server development

Product implementation in client hospitals via remote access

System maintenance

Triage of customer reported issues

 QUALIFICATIONS:

Bachelor degree in Mathematics, Computer Science, or related disciplines from a 4-year college or university. Graduate degree is a plus.

Four plus years software development experience

Thorough understanding of software development lifecycle models

Expertise in Java. C/C++ is a plus.

Programming experience in J2EE technology

Experience in Web UI development

Familiarity with JBoss application server and SEAM knowledge is a plus

Good knowledge and experience in Oracle database

Familiarity of network/communication protocol is also a plus

Good communication skills, both verbal and written

Ability to work successfully in small teams as well as independently to meet project deadlines
