DATABASE ADMINISTRATOR

GENERAL SUMMARY

The Database Administrator will maintain and support various business applications across the company.

ESSENTIAL JOB FUNCTIONS*

Oracle Database Administration:

Proactively monitor and manage potential issues thru GRID, scripts etc.

Patching, upgrading and other maintenance activities

Backups and recovery tasks thru RMAN

Database fine tuning and proactively monitor performance

Application Administration:

Maintain and support various business applications.

Besides, normal operational tasks, should handle time bound project related tasks such as staging new applications, upgrading, patching etc.

Proactively monitor potential issues thru the application seeded functionalities, scripting or thru other monitoring utilities.

OTHER DUTIES AND RESPONSIBILITIES

Document all the technical aspects related to Database and application administration.

Root Cause analysis and resolution for all database and application issues.

PREPARATION, KNOWLEDGE, SKILLS & ABILITIES

Bachelors or Masters’ Degree in a technical or business discipline. 2 + years as oracle DBA. Experience in SQL server administration is a plus.

3 + years’ experience in managing and administrating business application one or more major applications.

Demonstrated experience in working on root cause analysis for database and application issues.

Able to work independently and as a team player.

Flexible, adaptable, and able to manage multiple tasks in a dynamic environment

Able to learn new technologies and applications quickly

Excellent written and oral communication skills.

DBA certification is a plus
